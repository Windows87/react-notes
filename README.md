## How to run
First clone the repository:
```bash
# Clone this repository
git clone https://github.com/Windows87/react-notes
# Go into repository
cd react-notes
```

Start the react app:
```bash
# Install dependencies
npm install
# Start
npm start
```

Start NW.js:
```bash
# Install NWjs
npm install -g nw
# Start
nw .
```
